About This Project

Mini Task Scheduler is a lightweight OS project demonstrating scheduling concepts. Backend in C, frontend in HTML/CSS/JS.

  • Smart Workload Balancer
  • Productivity Score
  • Deadline Pressure Meter
  • Task Urgency Heatmap
  • Offline-first smart logging
  • Fullscreen mobile reminders
